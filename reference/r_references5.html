      In an information system, input is the raw data that is processed to produce output.

    • Markdown

      Markdown is a lightweight markup language for creating a formatted text using a plain-text editor. Markdown is widely used in blogging, instant messaging, online forums, collaborative software, documentation pages, and readme files.

    • Graphical elements

      Using graphics is an essential part of creating technical documentation. They help you convey complex information in a more understandable and clear way for the readers.
